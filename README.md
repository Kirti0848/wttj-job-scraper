# 🌐 WTTJ Job Scraper

Web scraper built for Relu Consultancy Data Extraction Engineer challenge.

## 🛠️ Tech Used
- Python
- Selenium
- Pandas
- ChromeDriver

## 📋 What it does
- Scrapes all "Business" jobs from Welcome to the Jungle (US)
- Handles pagination automatically
- Cleans data as per requirements
- Exports to CSV with 10 fields

## 📊 Results
- Total jobs scraped: 30
- Jobs in New York: 14
- Permanent Contract roles: 27

## ▶️ How to run
pip install selenium webdriver-manager
python wttj_scraper.py
